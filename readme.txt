Suivez et optimisez l'origine de vos appels entrants avec le Call-tracking Magnétis en intégrant des numéros trackés dans WordPress.

== Description ==

Analysez l'origine de vos appels entrants avec le Call-tracking Magnétis. Déterminez quels canaux vous génèrent le plus d’appels et optimisez votre budget de communication.
L'extension de Call-Tracking Magnétis intègre vos numéros trackés directement dans votre site Wordpress. Retrouvez ensuite vos statistiques d'attribution dans votre interface de call-tracking Magnetis.

Ainsi vous saurez :

*  si un visiteur vous appelle grâce à votre référencement naturel, Google Ads, vos réseaux sociaux, vos sites partenaires etc.
*  le nombre d'appels reçus
*  le temps moyen par appel
*  le nombre d'appels manqués ou non répondus
*  les canaux de communication les plus performants

Pour utiliser le call-tracking Magnétis, vous devez disposer d'un compte Magnétis actif. Vous pouvez créer votre compte gratuitement sur notre site [magnetis.fr](https://www.magnetis.fr/)

== Installation ==

Comment installer l'extension de call-tracking Magnétis ?

1. Téléchargez l'extension dans le dossier `/wp-content/plugins/magnetis-calltracking`, ou installez le à partir de la bibliothèque des extensions.
2. Activez l'extension à partir de l'écran Extension dans WordPress
3. Si vous n'avez pas encore de compte Magnétis, créez votre compte à partir de notre site [app.magnetis.fr](https://app.magnetis.io/register)
4. Dans votre compte Magnétis, récupérez votre clé module. Faites attention de bien choisir la clé dedié au module du site souhaité si vous possédez plusieurs modules.
5. Retournez dans votre interface Wordpress et dans "Réglages > Magnétis", collez votre clé module dans le champ dédié
6. Recevez vos appels et profitez de l’analyse

Pour plus d'informations, vous pouvez vous reporter à l'espace Support dans votre compte Magnétis

== Frequently Asked Questions ==

= Comment puis-je tester le call-tracking =

Vous pouvez créer votre compte gratuitement et demander un numéro tracké depuis votre interface.

= L'intégration de mon numéro tracké est-elle automatique =

Oui. L'extension Magnétis remplacera automatiquement dans votre site votre numéro habituel par le numéro tracké Magnétis. Vous devez au préalable configurer votre module web avec les paramètres nécessaires.

= Comment se passe la redirection du numéro tracké vers mon propre numéro =

La redirection est automatique et transparente. Il n’y a pas d’étapes supplémentaires ni pour vous ni pour l’appelant.
